# CounterX Typography

## Primary Font

**Inter**
- Weights: Regular (400), Medium (500), Semibold (600), Bold (700)
- Default for UI, docs, site, product, dashboards
- Open-source and optimized for digital interfaces

## Premium Alternatives (Optional)

- Satoshi
- GT America
- Suisse Int'l

## Hierarchy Standards

- **H1:** Inter Bold (700)
- **H2:** Inter Semibold (600)
- **H3:** Inter Medium (500)
- **Body:** Inter Regular (400)
- **Buttons:** Inter Medium (500) / Semibold (600)

## Letter-Spacing

- **Headings:** -1% to 0
- **Body text:** default
- **Buttons:** +1%
