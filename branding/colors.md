# CounterX Color Palette

## Primary Color

**Cobalt Trust**
- HEX: `#0044CC`
- RGB: `0, 68, 204`
- Usage: Logo, CTAs, highlights, key UI elements

## Core Neutrals

**Midnight Black**
- HEX: `#070708`
- RGB: `7, 7, 8`
- Usage: Text, strong contrast, dark themes, hero sections

**Soft Steel**
- HEX: `#9EA3B0`
- RGB: `158, 163, 176`
- Usage: Secondary text, subtle UI borders, muted elements

**Snow White**
- HEX: `#FFFFFF`
- RGB: `255, 255, 255`
- Usage: Clean background, maximum readability

## Usage Rules

- ❌ No gradients
- ✅ Bright blue only in solid blocks
- ✅ Reserve blue for highlights
- ✅ Light mode: lots of white space
- ✅ Dark mode: deep black + electric blue highlights
